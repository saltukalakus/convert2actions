# convert2actions

A static code conversion tool to convert Auth0 Rules/Hooks to Auth0 Actions 

## How to run the app locally

```
> npm install
> npm run
```

## Testing 

```
> npm test
```

## Support Features

# Rules to Actions

### Completed 
* Function name
* Callback functions
* User object
* Secrets
* Multi-factor
* Custom claims
